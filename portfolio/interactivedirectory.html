INTERACTIVE DIRECTORY


This is an interactive directory that was developed for the Dartmouth Computer Science Department. It includes key information about the department heads, professors, Ph.D students, etc. as well as a detailed map for navigation. Throughout the process, I managed the UI/UX and created the mockups for the user-facing portion. We created two mockups and ultimately chose to implement the flat color version.
